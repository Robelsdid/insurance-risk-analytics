# Insurance Risk Analytics Project

## Project Overview
This project is developed for AlphaCare Insurance Solutions (ACIS) to analyze historical insurance claim data for car insurance in South Africa. The analysis aims to optimize marketing strategy and identify "low-risk" targets for potential premium reduction to attract new clients.

## Project Objectives
1. Analyze insurance claim data to identify risk patterns
2. Perform A/B hypothesis testing on various demographic factors
3. Develop predictive models for optimal premium calculation
4. Generate actionable insights for marketing strategy optimization

## Project Structure
- `data/`: Contains raw and processed datasets
- `notebooks/`: Jupyter notebooks for exploratory data analysis and modeling
- `src/`: Source code for data processing, analysis, and modeling
- `reports/`: Generated reports and visualizations
- `docs/`: Documentation including insurance terminology glossary

## Key Analysis Areas
1. Insurance Terminology Research
2. A/B Hypothesis Testing
   - Risk differences across provinces
   - Risk differences between zipcodes
   - Margin differences between zipcodes
   - Risk differences between genders
3. Machine Learning & Statistical Modeling
   - Linear regression models for total claims prediction by zipcode
   - ML model for optimal premium prediction based on various features

## Final Deliverables
- Comprehensive analysis report with methodologies
- Findings from statistical tests and modeling
- Recommendations for insurance plan feature modifications
- Feature importance analysis for predictive models
